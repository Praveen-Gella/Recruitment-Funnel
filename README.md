# Recruitment-Funnel
Tracked stages: Application → Screening → Interview → Offer → Hire
Used Power BI dashboards and DAX measures like:
Cost per Hire
Time to Hire
Offer Acceptance Ratio (OAR)
Key insights:
1000 applicants → 328 hires
High OAR (0.93) = strong hiring efficiency
Best sources: LinkedIn & Referrals
Identified top & low-performing recruiters
